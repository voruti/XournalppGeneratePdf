# Xournalpp generate PDF

Quick access to (re-)generate PDFs out of [xournalpp/xournalpp](https://github.com/xournalpp/xournalpp) files.
Includes Windows wrapper script to call the bash script with Git SCM.

`xournalpp` needs to be in the path and the `GIT_BASH` env var needs to point to the `bash.exe` when used on Windows.
